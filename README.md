# 🧾 Inventory Management System (Flask Version)

A simple yet powerful **Inventory Management Web Application** built using **Python (Flask)** and **SQLite**, with responsive UI using HTML/CSS.  
This app allows you to **add, update, delete, search**, and **track stock levels** of products with ease.

---

## 🌟 Features

- ✅ **Responsive UI** for both desktop and mobile
- ✅ **Product entry form**
- ✅ **Product list table**
- ✅ **Add / Update / Delete** products
- ✅ **Low stock alert** for items with stock ≤ 5
- ✅ **Search** by product name or category
- ✅ **Modern look** with background and stylings
